## ðŸ’¡ Important Notes

### Image Optimization
- Compress images before uploading
- Use TinyPNG.com or similar
- Target sizes:
  - Hero: 1920x1080px (can be larger)
  - About: 600x400px
  - Menu items: 300x250px
  - Gallery: 600x600px (square)

### JSON Format
- Must be valid JSON (no trailing commas)
- Quote all keys and strings with double quotes
- Check syntax at jsonlint.com if unsure
- Indentation is optional but recommended

### File Naming
- Keep filenames lowercase
- Use hyphens for spaces (my-image.jpg)
- No special characters
- Match exactly in data.json
